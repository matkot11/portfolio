# Portfolio
Link to page: [portfolio](https://mateuszkocik.netlify.app)

### About
Portfolio page is made in Gatsby. Project info is stored in DatoCMS. Layout is made with Styled Components. I used emailJS to handle contact form.

### Technologies
* Gatsby
* DatoCMS
* GraphQL
* Styled Components
* emailJS

## To run website type: 

 `gatsby develop`
